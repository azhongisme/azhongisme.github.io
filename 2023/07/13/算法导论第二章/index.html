算法导论第二章

2023/7/13 algorithm

插入排序是算法导论第二章的第一个算法，适用于少量数据的排序，假设数组A[1….n-1]有序，则未排序的A[1….n]的第n个元素
即A[n]的位置在A[1….n-1]中插入

具体代码如下

include<bits/stdc++.h>
int main(){
  int n;
  std::cin>>n;
  std::vector<int>a(n);
  for(int i = 0;i<n;i++>){
      std::cin>>a[i];
  }
  for (int i = 1; i < n; i++) {
      int key = a[i];
      int j = i - 1;
      while (j >= 0 && a[j] > key) {
          a[j + 1] = a[j];
          j--;
      }
      a[j + 1] = key;
  }
  for (int i = 0; i < n; i++) {
      std::cout << a[i] << " ";
  }
}

算法分析 ：
 上述代码实现的升序排序，假设最糟糕的情况为原数组为降序，那么需要计算n(n+1)/2次
 因此时间复杂度为O(n²)

分治法 ：
 许多有用的算法在结构上是递归的: 为了解决一个给定的问题,算法一次或多次递归地调用其自身用其自身以解决紧密相关的
若干子问题。这些算法典型地遵循分治法地思想: 将原问题分解为几个规模较小但类似于原问题的子问题,递归地求解这些子问
题,然后再合并这些子问题的解来建立几个问题的解。
  分治模式在每次递归时都有三个步骤：
  分解原问题为若干子问
  解决子问题
  合并子问题的解构成原问题的解

归并排序(nlgn)：
 归并排序算法完全遵循分治模式
 分解 分解待排序的n个元素的序列成各具n/2个元素的子序列
 解决 使用归并排序递归地排序两个子序列
 合并 合并两个已排序的子序列以产生已排序的答案
代码如下 

#include<iostream>
#include<algorithm>
using namespace std;
int n;
const int N = 100010;
//用tmp数组存储子序列
int q[N],tmp[N];

void merge_sort(int q[],int l,int r){
  if(l>=r) return;
  
  int mid = l+r>>1;
  
  merge_sort(q,l,mid),merge_sort(q,mid+1,r);
  
  int k = 0,i = l,j = mid+1;
  
  while(i<=mid&&j<=r){
      if(q[i]<=q[j]) tmp[k++] = q[i++];
      else tmp[k++] = q[j++];
  }
  
  while(i<=mid) tmp[k++] = q[i++];
  while(j<=r) tmp[k++] = q[j++];
  
  for(i = l,j = 0;i<=r;i++,j++) q[i] = tmp[j];
}

int main()
{
  scanf("%d",&n);
  for(int i = 0;i<n;i++) scanf("%d",&q[i]);
  merge_sort(q,0,n-1);
  for(int i = 0;i<n;i++){
      printf("%d ",q[i]);
  }
}
