树形DP

2023/7/26 algorithm


树形DP其实和普通的DP问题大差不差，都涉及两个方面：状态表示和状态转移方程，只是运用到了树这种数据结构，
其中状态转移一般通过DFS或者BFS来实现。
下面以几道例题为例

Ural 大学有 N名职员，编号为 1∼N。

他们的关系就像一棵以校长为根的树，父节点就是子节点的直接上司。

每个职员有一个快乐指数，用整数Hi给出，其中 1≤i≤N
。

现在要召开一场周年庆宴会，不过，没有职员愿意和直接上司一起参会。

在满足这个条件的前提下，主办方希望邀请一部分职员参会，使得所有参会职员的快乐指数总和最大，求这个最大值。

输入格式
第一行一个整数 N。

接下来 N行，第 i行表示 i号职员的快乐指数 Hi。

接下来 N−1行，每行输入一对整数 L,K，表示 K是L的直接上司。

输出格式
输出最大的快乐指数。

数据范围
1≤N≤6000,−128≤Hi≤127

这道题的状态表示可以表示为f[u][0]和f[u][1],其中f[u][0]表示以u为根节点的不含u的子树所得的最大值，f[u][1]为以u为根含有u的子树最大值
状态转移过程：f[u][0] += max(f[子结点][0],f[子节点][1]),f[u][1] += f[子节点][0]
最终结果为 max(f[根节点][0],f[根节点][1])

完整代码如下：

#include<bits/stdc++.h>
using namespace std;

const int N = 6100;

bool father[N];
int f[N][2];
vector<int>happy(N);
vector<vector<int>>g(N);

void dfs(int root){
    f[root][1] += happy[root];
    for(int i = 0;i<g[root].size();i++){
        dfs(g[root][i]);
        f[root][0] +=max(f[g[root][i]][0],f[g[root][i]][1]);
        f[root][1] +=f[g[root][i]][0];
    }
}

int main(){
    int n;
    cin>>n;
    
    for(int i = 1;i<=n;i++) cin>>happy[i];
    
    for(int i = 0;i<n-1;i++){
        int k,l;
        cin>>k>>l;
        g[l].push_back(k);
        father[k] = true;
    }
    int root = 1;
    while(father[root]){
        root++;
    }
    dfs(root);
    cout<<max(f[root][0],f[root][1]);
}

鲍勃喜欢玩电脑游戏，特别是战略游戏，但有时他找不到解决问题的方法，这让他很伤心。

现在他有以下问题。

他必须保护一座中世纪城市，这条城市的道路构成了一棵树。

每个节点上的士兵可以观察到所有和这个点相连的边。

他必须在节点上放置最少数量的士兵，以便他们可以观察到所有的边。

你能帮助他吗？

输入格式
输入包含多组测试数据，每组测试数据用以描述一棵树。

对于每组测试数据，第一行包含整数 N
，表示树的节点数目。

接下来 N
行，每行按如下方法描述一个节点。

节点编号：(子节点数目) 子节点 子节点 …

节点编号从 0到 N−1，每个节点的子节点数量均不超过 10，每个边在输入数据中只出现一次。

输出格式
对于每组测试数据，输出一个占据一行的结果，表示最少需要的士兵数。

数据范围
0<N≤1500,一个测试点所有 N相加之和不超过 300650。


/*分析：这道题只需要满足每条边最少有一个端点上有士兵即可
状态表示f[u][0] 以u为根的子数且u点没有士兵，f[u][1]以u为根的子树且u有士兵
状态转移:f[u][0] += f[j][1] f[u][1] += min(f[j][0],f[j][1]) */
#include<bits/stdc++.h>
using namespace std;

const int N = 1510;
vector<vector<int>>g(N);
int n;
int f[N][2];
bool st[N];
void dfs(int u){
    f[u][1] = 1;
    for(int i = 0;i<g[u].size();i++){
        int j = g[u][i];
        dfs(j);
        f[u][0] += f[j][1];
        f[u][1] += min(f[j][0],f[j][1]);
    }
}

int main(){
    while(scanf("%d",&n)!=EOF){
        memset(st,0,sizeof(st));
        memset(f,0,sizeof(f));
        for(int i = 0;i<n;i++) g[i].clear();
        for(int i = 0;i<n;i++){
            int id,cnt;
            scanf("%d:(%d) ",&id,&cnt);
            while(cnt--){
                int a;
                cin>>a;
                g[id].push_back(a);
                st[a] = 1;
            }
        }
        int root = 0;
        while(st[root]){
            root++;
        }

        dfs(root);

        cout<<min(f[root][1],f[root][0])<<"\n";
    }
}

太平王世子事件后，陆小凤成了皇上特聘的御前一品侍卫。
皇宫以午门为起点，直到后宫嫔妃们的寝宫，呈一棵树的形状，某些宫殿间可以互相望见。大内保卫森严，三步一岗，五步一哨，每个宫殿都要有人全天候看守，
在不同的宫殿安排看守所需的费用不同。
可是陆小凤手上的经费不足，无论如何也没法在每个宫殿都安置留守侍卫。
帮助陆小凤布置侍卫，在看守全部宫殿的前提下，使得花费的经费最少。

输入描述:
输入中数据描述一棵树，描述如下：
第一行n，表示树中结点的数目。
第二行至第n+1行，每行描述每个宫殿结点信息，依次为：该宫殿结点标号,在该宫殿安置侍卫所需的经费k，该边的儿子数m，接下来m个数，分别是这个节点的m个儿子的标号
对于一个n个结点的树，结点标号在1到n之间，且标号不重复。

输出描述:
输出最少的经费

示例1
输入

6
1 30 3 2 3 4
2 16 2 5 6
3 5 0
4 4 0
5 11 0
6 5 0
输出
25

/*该题与上一题不同的是每个点所占的权重不同，因此不能只根据一条至少有一个端点有士兵得到最后的结果
该题存在可能一条边两个点都没有士兵占据
状态表示 f[u][0] u点没有士兵可以由父节点看到 f[u][1] u点没有士兵可以由子节点看到 f[u][2]该点有士兵
状态转移 f[u][0] += min(f[j][1],f[j][2]) f[j][1]表示u的子节点可以被子节点的子节点看到
        f[u][2] += min(f[j][2],min(f[j][0],f[j][1]))
        f[u][1] = min(f[u][1],sum+f[j][2]-min(f[j][1],f[j][2])) sum = f[u][0]
*/
#include<bits/stdc++.h>
using namespace std;

const int N = 1510;
int f[N][3];
vector<vector<int>>g(N);
int n;
bool st[N];
int w[N];

void dfs(int u){
    f[u][2] = w[u];
    for(int i = 0;i<g[u].size();i++){
        int j = g[u][i];
        dfs(j);
        f[u][0] += min(f[j][1],f[j][2]);
        f[u][2] += min(f[j][2],min(f[j][0],f[j][1]));
    }
    f[u][1] = 1e9;
    for(int i = 0;i<g[u].size();i++){
        int j = g[u][i];
        f[u][1] = min(f[u][1],f[j][2]+f[u][0]-min(f[j][1],f[j][2]));
    }
}


int main(){
    cin>>n;
    for(int i = 0;i<n;i++){
        int id,cnt,cost;
        cin>>id>>cost>>cnt;
        w[id] = cost;
        while(cnt--){
            int a;
            cin>>a;
            g[id].push_back(a);
            st[a] = 1;
        }
    }
    int root = 1;
    while(st[root]){
        root++;
    }
    dfs(root);
    //由于根节点没有父节点，所以没有f[root][0]
    cout<<min(f[root][1],f[root][2]);
    system("pause");
}
