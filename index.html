算法导论第二章
2023/7/13

插入排序是算法导论第二章的第一个算法，适用于少量数据的排序，假设数组A[1….n-1]有序，则未排序的A[1….n]的第n个元素
即A[n]的位置在A[1….n-1]中插入
具体代码如下

include<bits/stdc++.h>
int main(){
  int n;
  std::cin>>n;
  std::vector<int>a(n);
  for(int i = 0;i<n;i++>){
      std::cin>>a[i];
  }
  for (int i = 1; i < n; i++) {
      int key = a[i];
      int j = i - 1;
      while (j >= 0 && a[j] > key) {
          a[j + 1] = a[j];
          j--;
      }
      a[j + 1] = key;
  }
  for (int i = 0; i < n; i++) {
      std::cout << a[i] << " ";
  }
}

算法分析 ：
上述代码实现的升序排序，假设最糟糕的情况为原数组为降序，那么需要计算n(n+1)/2次
因此时间复杂度为O(n²)
